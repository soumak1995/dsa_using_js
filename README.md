# DSA — Problem Index

This repository contains JavaScript implementations of common data-structures & algorithms interview problems. Below is a list of problems with links to their implementations.

## Problems

- [Array_1/longest-sub-array-sum.js](Array_1/longest-sub-array-sum.js) — Find the length of the longest subarray with sum equal to K.
- [Array_1/majority-element.js](Array_1/majority-element.js) — Find the majority element appearing more than ⌊n/2⌋ times (Boyer–Moore).
- [dp/grid-with-maze-obstacles.js](dp/grid-with-maze-obstacles.js) — Unique Paths II: count unique paths in a grid with obstacles.
- [dp/climbing-stairs.js](dp/climbing-stairs.js) — Count distinct ways to climb `n` stairs (1 or 2 steps).
- [dp/frog-jump-II.js](dp/frog-jump-II.js) — Minimum energy to reach end with jumps up to `k`.
- [dp/house-robber.js](dp/house-robber.js) — Maximize robbery amount without robbing adjacent houses.
- [dp/max-sum-non-adjacent.js](dp/max-sum-non-adjacent.js) — Maximum sum with no two selected elements adjacent.
- [dp/grid-min-path-sum.js](dp/grid-min-path-sum.js) — Minimum path sum from top-left to bottom-right in a grid.
- [dp/ninja-training.js](dp/ninja-training.js) — Maximize points over days without repeating last task.
- [greedy/n-meeting-in-one-room.js](greedy/n-meeting-in-one-room.js) — Select maximum number of non-overlapping meetings.
- [Binary Tree/pre-order-traversal.js](Binary%20Tree/pre-order-traversal.js) — Preorder traversal of a binary tree (root-left-right).
- [sliding_window/fruit-into-baskets.js](sliding_window/fruit-into-baskets.js) — Max number of fruits picked consecutively with at most 2 types.
- [sliding_window/longest-repeating-char-replacement.js](sliding_window/longest-repeating-char-replacement.js) — Replace up to `k` chars to maximize repeating-character substring.
- [sliding_window/substr-contain-all-three-char.js](sliding_window/substr-contain-all-three-char.js) — Count substrings containing at least one of 'a', 'b', and 'c'.
- [sliding_window/longest-substring-with-most-k-dis-char.js](sliding_window/longest-substring-with-most-k-dis-char.js) — Longest substring with at most `k` distinct characters.
- [sliding_window/max-consecutive-one-III.js](sliding_window/max-consecutive-one-III.js) — Flip up to `k` zeros to maximize consecutive 1s.
- [sliding_window/longest-substring-without-repeating.js](sliding_window/longest-substring-without-repeating.js) — Length of longest substring without repeating characters.
- [Binary Tree/in-order-treversal.js](Binary%20Tree/in-order-treversal.js) — Inorder traversal of a binary tree (left-root-right).
- [Binary Tree/post-order-traversal.js](Binary%20Tree/post-order-traversal.js) — Postorder traversal of a binary tree (left-right-root).
- [greedy/job-sequencing.js](greedy/job-sequencing.js) — Schedule jobs to maximize profit given deadlines.
- [greedy/max-platform.js](greedy/max-platform.js) — Minimum number of platforms required for given arrival/departure times.
- [greedy/fractional-knapsack.js](greedy/fractional-knapsack.js) — Fractional knapsack: maximize value with fractional items allowed.
- [Array_1/remove-duplicate.js](Array_1/remove-duplicate.js) — Remove duplicates from sorted array (return unique portion).
- [dp/grid-unique-paths.js](dp/grid-unique-paths.js) — Count unique paths in an m × n grid (right & down moves).
- [Array_1/intersection.js](Array_1/intersection.js) — Intersection of two sorted arrays (unique common elements).
- [Array_1/max_subarray_sum.js](Array_1/max_subarray_sum.js) — Maximum subarray sum (Kadane's algorithm).
- [miscellaneous/trapping-rain-water.js](miscellaneous/trapping-rain-water.js) — Compute trapped rain water given elevation map.
- [Array_1/short-array-0-1-2.js](Array_1/short-array-0-1-2.js) — Sort array of 0s,1s,2s in-place (Dutch National Flag).
- [string/longest-common-prefix.js](string/longest-common-prefix.js) — Longest common prefix among an array of strings.
- [string/longest-pelindrome.js](string/longest-pelindrome.js) — Longest palindromic substring in a string.
- [string/reverse-every-word.js](string/reverse-every-word.js) — Reverse characters of every word in a string.
- [string/valid-anagram.js](string/valid-anagram.js) — Check if two strings are anagrams of each other.
- [Array_1/union-two-poniter.js](Array_1/union-two-poniter.js) — Union of two sorted arrays (unique sorted elements).
